# 🏛️ eCourts Scraper Intern Task – Flask UI Demo

This project was created as part of the **eCourts Scraper Internship Assignment**.  
It demonstrates how to fetch cause lists from the eCourts portal and download them as PDFs using a real-time user interface.

---

## 🚀 Features
- Dynamic **UI built with Flask + Bootstrap**
- Select **State → District → Court Complex → Court Name → Date**
- Automatically generates **Cause List PDF** for selected date
- Works **instantly in real time** (demo-safe version without external scraping)
- Clean JSON structure for easy data use
- Ready for further integration with live eCourts API

---

## 🧠 Tech Stack
- **Python 3**
- **Flask** – web framework  
- **BeautifulSoup4** – HTML parsing  
- **ReportLab** – PDF generation  
- **Bootstrap 5** – responsive UI
